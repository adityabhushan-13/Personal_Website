Project-2: Weather Website

In this project, I have build a website that will fetch weather information from an API with latitude,longitude query if the user give their location access else the user can search for their city by their city names. Also, This website will give information like sunrise time, sunset time, humidity,visibility, pressure, wind speed, wind degree, minimim temperature, maximum temperature, feels like and weather description. Background image is also added using API by using latitude,longitude and city name as query.
